About the Open Siddur client app

The Open Siddur Project intends to create a complete end to end system for producing high quality Jewish prayer books, for paper publication or sharing online. We are committed to our Jewish tradition, free culture and open source.

Source code for the client and server are available from our github repositories. Unless otherwise indicated, code is licensed under the GNU Lesser General Public License, version 3 or later. Libraries may be released under other open source licenses. See the source code for details. Content may be licensed under any of the Creative Commons free culture licenses.

To report issues with this app or to get involved in the technical aspects of the project, contact the opensiddur-tech Google group.

Documentation

Client documentation and howtos are being collected on the project's github wiki.

About this app

This is version 0.2.0. It supports:

  • user registration and editing of first and third party user profiles
  • a bibliography editor
  • an outline editor
  • a basic XML editor for JLPTEI original texts, with dialogs for setting document metadata
  • a second, simple editor for limited parts of JLPTEI original texts
  • very basic sharing permissions: public or restricted
  • improved alignment of two parallel texts, such as translations, to link them together
  • inline editing of annotations in the simple editor and direct editing of annotations documents in the XML editor
  • inline settings and conditionals in the simple editor
  • list support in the simple editor
  • divisions with headers in simple editor
  • poetic verse divisions in simple editor
  • anchors in the simple editor
  • show included texts inline
  • basic editing of conditionals documents
  • editing of conditional documents inside text documents by UI (new in this version)
  • basic editing of styling documents
  • automatic retrieval of transcription images from sites that support embedding
  • dialog boxes to help insertion of intra- and inter-document links
  • showing recent changes
  • showing compilation status
  • showing active compilation jobs
